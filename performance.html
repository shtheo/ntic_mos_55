Performance

Pour performer, l'athlète essaye d'optimiser son entrainement, son style de vie, son alimentation. Il s'agit souvent d'un coach ou d'une équipe qui observe et qui prend des décisions sur comment l'entraînement doit se passer. L'idée que nous allons développer ici, c'est comment l'athlète peut devenir meilleur grâce à la technologie.

Mesurer ses performance : boom de l'IOT

Pour qu'une IA prenne des décisions, il lui faut des données. Dans le sport, le nombre d'outils de mesure a explosé grâce à l'IOT. En effet, depuis le début des années 2010, c'est le boom des objets connectés. Avec le boom du running depuis la fin des années 2000, tout le monde a entendu parler des montres connectées ou des réseaux sociaux comme Strava permettant de partager les courses effectuées. Mais le running n'est pas le seul sport dans lequel l'IOT est omniprésent. A l'heure actuelle, ces objets connectés se sont tellement développés qu'il est possible d'en trouver dans tous les sports. Voici une liste non exhaustive des objets connectés pour le sport :

  • La raquette de badminton connectée
  • Un nouveau suivi pour le skieur
  • Le robot ramasseur de balle de tennis
  • La chaussure de running connectée
  • Combinaison de patinage connectée
  • Ballon de football connecté

Vrai outil ou simple gadget ?

Pour beaucoup de sportifs amateurs, l'utilisation de ces objets connectés s'arrête là. Ce sont plus des gadgets pour avoir des chiffres sur la performance qu'un réel outil pour les améliorer. En effet, très souvent ces objets sont vendu tels quels. Ils recueillent les données avec l'objet, proposent un couche software pour les présenter à l'utilisateur, mais il n'y a pas de couche intelligente qui permette de traiter ces données efficacement dans un but de progrès.

Des exemples de réussite du Big Data dans le sport existent déjà. Prenons l'exemple l'équipe d'Allemagne de Football qui a gagné la coupe du monde en 2014. Pour cet évenement, un dispositif exceptionnel avait été mis en place grâce à leur partenariat avec SAP. Les joueurs étaient tous équipés de protège-tibias connectés. Cela permettait de récupérer des données durant les entraînements et les matchs. Ces données étaient ensuite couplées avec les statistiques classiques de chaque matchs. A partir de toutes ces données, SAP avait developpé un software reposant sur une IA. Elle permettait de voir quels joueurs présentaient des signes de fatigue plus avancés que d'autres, quels joueurs étaient plus à l'aise dans quel système de jeu, ... Cependant, il reste difficile de savoir quel facteur cette technologie a joué dans leur victoire finale. Le futur viendra avec d'autres exemples similaires. L'étude de ces exemples permettra de savoir quel rôle ces technologies ont réellement joué dans les succès sportifs.
